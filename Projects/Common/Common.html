Introduction

This project contains code common to all projects built under this test framework, such as array abstractions and testing code. Note that this project directly relies on the Accelerator project and so will build that first automatically.

Build Options

Note that not all of these options have been fully tested in all combinations. If you find a problem combinations please inform Alex Cole (t-alcole). Also note that you will need to rebuild "Common" with the same parameters to use these properly.

  • NO_ACCELERATOR - Exclude the Accelerator target from the build.
  • NO_ACCELERATOR_OPT - Exclude the optimised Accelerator target from the build.
  • NO_CUDA - Exclude the CUDA target from the build.
  • NO_CUDA_OPT - Exclude the optimised CUDA target from the build.
  • NO_OPEN_CL - Exclude the OpenCL target from the build.
  • NO_REFERENCE - Exclude the Reference (CPU) target from the build.
  • NO_REFERENCE_OPT - Exclude the optimised Reference (CPU) targets (most have two versions) from the build.
  • NO_STATIC_DATA - When generating arrays of input data, a number of methods can be used, including random, 0s and pre-set data. Using the pre-set data may make the executable huge, so can be excluded using this switch. Note that the inclusion of this data is the main reason for "Common" being a separate project to reduce regular build times.
